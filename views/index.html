Movie Recommendation System

Discover movies based on your personal preferences

Project Overview

This web application demonstrates a simple Movie Recommendation System built with Node.js and Express.js. Users can navigate through pages and submit their preferences using a contact form.

Features

  • Genre-based recommendations
  • Mood-based suggestions
  • Search by release year
  • Simple and clean UI
