Movie Recommendation System

We created a web app that suggests movies based on user preferences

Team Info

  • Name: Yerzhan Saniya, Zhanatay Ismail, Baizakova Erkezhan
  • Group: SE-2423

About the Project

This project is a basic Node.js + Express.js application that allows users to input movie preferences such as genre, release year, or mood. The system will then provide movie or TV show recommendations.

Planned Features

  • Genre-based movie recommendation
  • Mood-based movie suggestion
  • Search movies by release year
  • Visual UI improvements
  • API integration (future work)
  • View brief information about each movie
  • Receive personalized recommendations
  • Save movies they like
  • User profile
  • Maybe will add something more
